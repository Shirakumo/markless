\definesubsubsection{Font}
\defineidentifier{compound-font}{font <font .+>}
\definestyle{compound-font}{font-family: font} \\

If given, this option marks the \g{style} to change the font family. If the specified font is not available to the \g{user} for one reason or another, either no font change occurs, or an \g{error} is \glink{signalling}{signalled}. The \g{implementation} may make an effort to include the font in the \g{document} in such a way that it is not necessary for the user to have a copy of the font, but it is not required to. \\

\begin{examples}
  \longexample{"Comic sans"(font Comic Sans Ms) is a good font to annoy people.}{{\fontspec{Comic Sans MS}\selectfont Comic sans} is a good font to annoy people.}
\end{examples}
